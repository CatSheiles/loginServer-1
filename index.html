Login Server


Server Methods:

  • GET / - returns this webpage

  • POST /register - register new account
    • {usr: 'username', password: 'hashed password', salt:'password salt'}

  • POST /salt - get the salt value for a particular user
    • {usr: 'username'}

  • POST /login - logs a user in to an existing account
    • {usr: 'username', password:'Hased and salted password'}
